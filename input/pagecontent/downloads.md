### Downloads

This IG ships its conformance resources as a FHIR NPM package. To use the resources locally with the official FHIR validator (or any tool that reads the FHIR package format), install the package:

```
fhir install be.fhir.bcr 0.1.0
```

The full package archive can also be downloaded directly:

- [FHIR NPM package (.tgz)](package.tgz)
- [JSON definitions](definitions.json.zip)
- [XML definitions](definitions.xml.zip)
- [TTL definitions](definitions.ttl.zip)
- FSH source — see the project repository
